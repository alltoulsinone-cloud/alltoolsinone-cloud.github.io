دليل تأمين شبكة الواي فاي المنزلية بشكل محترف

كشف المتسللين، منع سرقة النت، وحماية بياناتك الشخصية

لماذا يجب تأمين شبكتك المنزلية؟

شبكة الواي فاي غير المؤمنة هي بوابة مفتوحة للمتسللين ل:

  • سرقة بياناتك الشخصية
  • استخدام إنترنتك مجاناً
  • شن هجمات من خلال IP الخاص بك
  • التجسس على اتصالاتك
  • اختراق الأجهزة المتصلة (كاميرات، هواتف، أجهزة ذكية)

1 تغيير بيانات الدخول الافتراضية للراوتر

أول خطوة وأهمها. معظم المستخدمين يتركون اسم المستخدم وكلمة المرور الافتراضية.

كيفية التغيير:

  1. ادخل لإعدادات الراوتر (عادة 192.168.1.1 أو 192.168.0.1)
  2. استخدم admin/admin أو admin/password للدخول أول مرة
  3. اذهب إلى System أو Administration
  4. غير اسم المستخدم وكلمة المرور

2 تحديث برنامج الراوتر (Firmware)

التحديثات تصلح ثغرات أمنية معروفة.

خطوات التحديث:

  1. ابحث عن طراز الراوتر الخاص بك
  2. اذهب لموقع الشركة المصنعة
  3. حمل أحدث إصدار
  4. ادخل إعدادات الراوتر → System Update
  5. اختر الملف المحمل وقم بالتحديث

3 تفعيل تشفير WPA3 أو WPA2

تأكد أن تشفير الشبكة هو WPA2 على الأقل، والأفضل WPA3 إذا متوفر.

تجنب:

  • WEP: قديم وسهل الاختراق (يكسر في 5 دقائق)
  • WPA: غير آمن أيضاً
  • فتح (Open): كارثة أمنية

4 اختيار كلمة مرور قوية للواي فاي

يجب أن تكون كلمة المرور:

  • 12 حرفاً على الأقل
  • مزيج من أحرف كبيرة وصغيرة
  • تحتوي على أرقام ورموز خاصة
  • لا تكون معلومات شخصية (هاتف، تاريخ ميلاد)
  • مثال: Meknes@Tech_2024#Secure

أدوات للكشف عن المتسللين

1. باستخدام Termux (على الهاتف)

2. باستخدام Wireless Network Watcher (ويندوز)

5 تفعيل MAC Address Filtering

السماح للأجهزة المعروفة فقط بالاتصال.

كيفية معرفة MAC Address:

# على ويندوز: ipconfig /all # على Linux/Termux: ifconfig # على Android: # الإعدادات → حول الهاتف → حالة → عنوان MAC للواي فاي

6 إخفاء SSID (اسم الشبكة)

شبكتك لن تظهر في قائمة الشبكات المتاحة، يجب كتابة اسمها يدوياً للاتصال.

7 تقليل قوة الإرسال (Tx Power)

تقليل مدى الإشارة لمنع الوصول من خارج المنزل.

كشف ومكافحة هجمات الواي فاي

هجوم Evil Twin

يقوم المهاجم بإنشاء شبكة واي فاي بنفس اسم شبكتك لجذب الضحايا.

الحماية:

  • لا تتصل بشبكات واي فاي عامة مجهولة
  • استخدم VPN عند الاتصال بشبكات عامة
  • تفقد عنوان MAC للشبكة

هجوم Deauthentication

يقطع المهاجم اتصالك بالشبكة لإجبارك على إعادة الاتصال وسرقة بياناتك.

الكشف باستخدام Termux:

pkg install aircrack-ng airodump-ng wlan0 # مراقبة حركة الشبكة

سكريبت مراقبة شبكة باستخدام Python

# network_monitor.py import scapy.all as scapy import time def scan_network(ip_range): print(f"جاري مسح الشبكة {ip_range}") arp_request = scapy.ARP(pdst=ip_range) broadcast = scapy.Ether(dst="ff:ff:ff:ff:ff:ff") packet = broadcast/arp_request answered = scapy.srp(packet, timeout=1, verbose=False)[0] devices = [] for element in answered: device_info = { "ip": element[1].psrc, "mac": element[1].hwsrc } devices.append(device_info) return devices def monitor_network(known_devices, ip_range="192.168.1.0/24"): while True: current_devices = scan_network(ip_range) unknown_devices = [] for device in current_devices: if device["mac"] not in [d["mac"] for d in known_devices]: unknown_devices.append(device) if unknown_devices: print("⚠️ تحذير: أجهزة غير معروفة على الشبكة!") for device in unknown_devices: print(f" IP: {device['ip']} | MAC: {device['mac']}") time.sleep(60) # فحص كل دقيقة # قائمة بالأجهزة المعروفة known_devices = [ {"ip": "192.168.1.2", "mac": "AA:BB:CC:DD:EE:FF"}, {"ip": "192.168.1.3", "mac": "11:22:33:44:55:66"} ] monitor_network(known_devices)

أدوات مجانية للحماية

  • Wireshark: تحليل حركة الشبكة
  • Nmap: اكتشاف الأجهزة والخدمات
  • GlassWire: مراقبة استخدام الشبكة
  • Bitdefender Home Scanner: فحص الشبكة المنزلية

نصائح إضافية للحماية

  1. أطفئ الراوتر عندما تخرج لفترة طويلة
  2. افصل الأجهزة الذكية غير المستخدمة
  3. استخدم شبكة ضيوف (Guest Network) للزوار
  4. غير كلمة المرور كل 3-6 أشهر
  5. راقب فاتورة الإنترنت للكشف عن استخدام غير طبيعي
