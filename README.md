# Tech Meknes – تعليم تقني

مرحبا بيك في **Tech Meknes**  
موقع تعليمي مجاني يركز على:

- الأمن السيبراني والحماية
- أدوات Android (Termux، root، scripts)
- الشبكات والراوترات
- DVR وكاميرات مراقبة (Hikvision، Imou...)
- نصائح عملية وتجارب حقيقية

### لماذا هاد الموقع؟
هنا ما غاديش نعطيوك نظريات فقط — غادي نشاركو معاك أدوات، سكريبتات، تجارب، وحلول حقيقية من الشارع.

### كيفاش تساهم؟
- اقترح موضوع جديد (Issues)
- أرسل تعديل أو مقالة (Pull Request)
- شارك الموقع مع اللي كيحب التقنية

ابقى معانا — المحتوى جاي قريب!
